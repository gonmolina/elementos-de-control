Error de estado estacionario y sistema tipo#

El error \(e(t)\) de un sistema lo definido como la referencia menos la salida, es decir:

\[e(t)=r(t)-y(t)\]

o lo que es lo mismo, en el dominio de Laplace:

\[E(s)=R(s)-Y(s)\]

En la figura mostramos dos diagramas gen칠ricos de bloques equivalentes, con la definici칩n del error. Observemos del segundo diagrama de bloques, que si \(H/I = 1, e(t) = a(t)\).

Dicho error tendr치 un transitorio, y luego llegar치 a un estacionario al cual se lo denomina error de estado estacionario.

Caso especial: Realimentaci칩n unitaria y constantes de error#

Si \(H = I\), podemos modelar el sistema con realimentaci칩n unitaria como muestra la siguiente figura.

Entonces tendremos:

\[F(s) = \dfrac{E(s)}{R(s)} = \dfrac{1}{1+G(s)}\]

Consideraremos a las entradas \(r(t)\) polin칩micas de la forma

\[r(t) = \dfrac{t^k}{k!} \text{ para } t>0\]

siendo su transformada de Laplace

\[R(s)=\dfrac{1}{s^{k+1}}\]

Se toma como base un sistema mec치nico gen칠rico como referencia para la nomenclatura, llamando para las entradas con \(k=0\) entrada de posici칩n, para \(k=1\) entrada de velocidad, y para \(k=2\) entrada de aceleraci칩n, sin tener en cuenta las unidades de la se침al de entrada actual. \[\begin{split} \begin{align} \lim_{t\rightarrow \infty} e(t) & = e_{ss}= \lim_{s\rightarrow 0} sE(s)\\ & = \lim_{s\rightarrow 0} \frac{1}{1+G}R(s)\\ & = \lim_{s\rightarrow 0} s \frac{1}{1+G}\frac{1}{s^{k+1}} \end{align} \end{split}\]

Consideremos primero un sistema para donde \(G(s)\) no tiene polo en el origin, es decir no un ning칰n integrador en \(G(s)\) y que se tiene una entrada escal칩n \(R(s)=\dfrac{1}{s}\). De esta manera \(r(t)\) es una entrada polinomial de grado 0. En este caso podemos escribir las ecuaciones anteriores:

\[\begin{split} \begin{align} e_{ss} & = \lim_{s\rightarrow 0} s\frac{1}{1+G(s)}\frac{1}{s}\\ \dfrac{e_{ss}}{r_{ss}} & = \dfrac{e_{ss}}{1}=e_{ss} = \frac{1}{1+G(0)} \end{align} \end{split}\]

Si \(G(s)\) tiene un polo en el origen, entonces podr칤amos continuar esta linea argumentativa y considerar un polinomio de orden 1 como entrada \(r(s)\), que es directo evaluarlo directamente de manera general. Para este caso es necesario describir el comportamiento de la planta y el controlador cuando \(s\) se aproxima a 0. Para este prop칩sito agrupamos todos los t칠rminos excepto los polos en el origen de la funci칩n \(G(s)\) en la funci칩n \(G_0(s)\), de manera que podamos definir \(G_0(0)=K_n\) y podemos escribir la funci칩n \(G(s)\) como:

\[G(s) = \frac{G_0(s)}{s^n}\]

De estar forma, si por ejemplo \(G(s)\) no tienen integradores, entonces \(n=0\); si tiene 1 integrador entonces \(n=1\), y as칤 sucesivamente. Sustituyendo esto en la expresi칩n del error:

\[\begin{split} \begin{align} e_{ss} & = \lim_{s\rightarrow 0 } s\frac{1}{1+\frac{G_0(s)}{s^n}}\frac{1}{s^{k+1}}\\ & = \lim_{s\rightarrow 0} \frac{s^n}{s^n+k_n}\frac{1}{s^k} \end{align} \end{split}\]

Para esta ecuaci칩n podemos ver que si \(n> k\) entonces \(e =0\) y si \(n<k\) entonces el error \(e(t) \rightarrow \infty\).

Si \(n = k = 0\) entonces \(e_{ss} = \dfrac{1}{1+K_0}\) y si \(n = k \neq 0\) entonces \(e_{ss} = \frac{1}{K_n}\).

Como se dijo anteriormente si \(n= k =0\), la entrada es una se침al polinomial de orden 0 y la constante \(K_p\) se llama constante de posici칩n, se la escribe \(K_p\) y el sistema se cl치sifica como sistema de Tipo 0.

Un sistema tambi칠n puede ser clasificado seg칰n su habilidad para rechazar perturbaciones de forma polin칩mica de manera an치loga al esquema de clasificaci칩n hecho para el seguimiento a referencias.

Supongamos ahora que tenemos un sistema de la forma que se ve en la siguiente figura (notar el cambio de notaci칩n):

Si queremos analizar como se comporta el error del sistema frente a una perturbaci칩n \(W(s)\), vamos a pensar que la referencia \(r(t)\) se mantiene en 0. Adem치s para este curso no se va a tener en cuenta el ruido de medici칩n \(v(t)\). As칤 \(V(s)=0\) y \(R(s)=0\).

As칤 tenemos que:

\[E(s)= -Y(s)\]

Nos interesa la funci칩n transferencia:

\[\frac{E(s)}{W(s)}=\frac{-Y(s)}{W(s)} = -\frac{G(s)}{1+G(s)D_{cl}} = T_w(s)\]

debido a que la entrada es igual a 0, la salida es el error (con el signo cambiado). De manera an치loga que para las entradas referencias, se dice que el sistema es Tipo 0 si para un perturbaci칩n escal칩n un sistema resulta con error constante distinto de 0 en estado estacionario; Tipo 1 si para una rampa el error resulta constante distinto de 0 y as칤 sucesivamente. En general, siguiendo el mismo enfoque usando para las entradas referencias, asumimos que una constante \(n\) y una funci칩n \(T_{o,w}(s)\) puede ser definida con las propiedades que:

  • \(T_{o,w}(0) = \dfrac{1}{K_{n,w}}\)

  • \(T_w(s) = s^nT_{o,w}(s)\)

Entonces el error en estado estacionario a una entrada perturbaci칩n que sea polinomial de grado \(k\) es:

\[\begin{split} \begin{align} y_{ss} & = \lim_{s\rightarrow 0 } \left[ sT_w(s)\frac{1}{s^{k+1}}\right]\\ & = \lim_{s\rightarrow 0} \left[ T_{o,w}(s)\frac{s^n}{s^{k}}\right] \end{align} \end{split}\]

De ac치 tenemos que se \(n>k\) el error es 0 y si \(n<k\) el error no es acotado. Si \(n = k\) el sistema es de tipo \(k\) y el error es dado por \(\dfrac{1}{K_{n,w}}\).
